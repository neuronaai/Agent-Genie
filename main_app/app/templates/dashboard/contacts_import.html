CSV Format Guide

Your CSV file must include a phone_number column. All other columns are optional.

Column Required Description
phone_numberYesE.164 format (+14155551234) or 10-digit US number
first_nameNoContact's first name (injected as LLM variable)
last_nameNoContact's last name
emailNoContact's email address
timezoneNoIANA timezone (auto-detected from area code if omitted)
any otherNoStored as dynamic data and injected into LLM variables
